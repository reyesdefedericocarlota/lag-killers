# Lag Killers

Plataforma musical full stack para crear playlists y gestionar canciones.

# Instrucciones de arranque

## Backend

### 1. Entrar en la carpeta backend

```bash
cd backend
```

### 2. Ejecutar Spring Boot

#### Windows

```bash
mvnw spring-boot:run
```

#### Mac / Linux

```bash
./mvnw spring-boot:run
```

### 3. Acceder al backend

```text
http://localhost:8080
```

---

# Consola H2

URL:

```text
http://localhost:8080/h2-console
```

## Configuración H2

| Campo     | Valor                |
| --------- | -------------------- |
| JDBC URL  | `jdbc:h2:mem:testdb` |
| User Name | `sa`                 |
| Password  | _(vacío)_            |

---

# Frontend

## 1. Entrar en la carpeta frontend

```bash
cd frontend
```

## 2. Instalar dependencias

```bash
npm install
```

## 3. Ejecutar Angular

```bash
ng serve
```

## 4. Acceder al frontend

```text
http://localhost:4200
```
